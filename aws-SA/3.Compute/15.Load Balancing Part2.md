# ⚖️ Which Load Balancer Supports What?

| Load Balancer | Cross-Zone              |
| ------------- | ----------------------- |
| ALB           | ✅ Always ON             |
| CLB           | Optional                |
| NLB           | Optional (cost applies) |
| GWLB          | Optional (cost applies) |

👉 Important:

* ALB → cannot disable 
* NLB/GWLB → enabling may incur cost 

---

# 🧠 Decision Rule

```text
Need even load → Enable cross-zone
Need AZ isolation → Disable
```

# 🔄 3) Routing Algorithms

---

## 🔹 Least Outstanding Requests (Smart)

👉 Sends request to:

* Least busy instance

---

## 🔹 Round Robin (Simple)

👉 Sends:

* One by one in order

---

## 🔹 Flow Hash (NLB only)

👉 Based on:

* IP + port + protocol

👉 Same connection → same target

---

# ⚖️ Comparison

| Algorithm         | Behavior             |
| ----------------- | -------------------- |
| Least Outstanding | Smart load balancing |
| Round Robin       | Equal rotation       |
| Flow Hash         | Connection-based     |

---

# 🎯 Exam Traps

👉 “Uneven number of instances across AZ”
→ Enable cross-zone

---

👉 “Reduce cross-AZ cost”
→ Disable cross-zone

---

👉 “User session must stay same instance”
→ Sticky sessions

---

👉 “Instance with least load should receive traffic”
→ Least Outstanding Requests

---

👉 “Same TCP connection must hit same target”
→ Flow Hash (NLB)

---

# 🧩 Final Mental Model

```text
Cross-zone → fairness
Sticky → consistency
Algorithm → efficiency
```

---

# 🔥 Real-World Insight

* ALB → always cross-zone (you don’t worry)
* NLB → think about cost vs performance
* Sticky sessions → avoid unless required
