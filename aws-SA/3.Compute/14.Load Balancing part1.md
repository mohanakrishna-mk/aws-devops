---

# ⚡ Important Concept: Internal vs External

* **Internet-facing LB** → public
* **Internal LB** → private (inside VPC)

---

# 🌍 NLB Zonal vs Regional (Advanced Exam Topic)

👉 Regional DNS:

* Returns multiple IPs (all AZs)

👉 Zonal DNS:

* Returns 1 IP (specific AZ)

---

## Why use zonal?

* Reduce latency
* Avoid cross-AZ cost



---

# ⚖️ Quick Decision Guide

```text id="d3x5mz"
HTTP app → ALB
TCP/UDP → NLB
Security inspection → GWLB
Legacy → CLB
```

---

# 🎯 Exam Traps

❗ “Path-based routing”
→ ✅ ALB

---

❗ “Static IP required”
→ ✅ NLB

---

❗ “Inspect traffic with firewall”
→ ✅ GWLB

---

❗ “Multiple SSL certs”
→ ✅ ALB (not CLB)

---

# 🧩 Final Mental Model

```text id="kz6v9c"
ALB = smart routing
NLB = fast routing
GWLB = security routing
CLB = outdated
```
