# 🌐 2) Internet Access from VPC Lambda

👉 This is where most people get tricked.

## ❌ Myth:

“Put Lambda in public subnet → gets internet”

👉 WRONG

---

## ✅ Correct:

```text id="n5c3rd"
Lambda (private subnet) → NAT Gateway → Internet Gateway → Internet
```

👉 Must use:

* NAT Gateway / NAT Instance

---

## 🔑 Key Rule

```text id="2jz9rd"
Lambda in VPC = NO internet unless NAT
```

---

# ⚡ 3) Access AWS Services (Better Way)

Instead of NAT:

👉 Use VPC Endpoints

Example:

* Amazon DynamoDB endpoint

👉 Benefits:

* No internet
* More secure
* Lower cost

---

# 🌍 4) Fixed IP for Lambda (Exam Favorite)

👉 Default:

* Random public IP

👉 Problem:

* Cannot whitelist

---

## ✅ Solution

```text id="7pjc8n"
Lambda → NAT Gateway (Elastic IP) → Internet
```

👉 Now:

* External system sees **fixed IP**

---

# 🔁 5) Invocation Types (VERY IMPORTANT)

---

## 🔹 Synchronous

👉 Caller waits for response

Examples:

* Amazon API Gateway
* SDK / CLI

```text id="v1t49d"
Client → Lambda → Response
```

👉 Error handling = client side

---

## 🔹 Asynchronous

👉 Fire-and-forget

Examples:

* Amazon S3
* Amazon SNS
* Amazon EventBridge

---

### Behavior:

* Auto retries (up to 3 times)
* Can fail multiple times

👉 Must design for:

* **Idempotency** 

---

# 💀 6) Dead Letter Queue (DLQ)

👉 Where failed events go

Options:

* Amazon SQS
* Amazon SNS

# ⚖️ Key Comparison

| Feature          | SNS → Lambda | SQS → Lambda |
| ---------------- | ------------ | ------------ |
| Speed            | Immediate    | Delayed      |
| Reliability      | Lower        | Higher       |
| Scaling          | Explosive    | Controlled   |
| Failure handling | Weak         | Strong       |

---

# 🧠 Final Decision Rule

```text id="z5h6fd"
Need speed → SNS
Need reliability → SQS
```

# 🎯 Exam Traps

👉 “Lambda needs to access RDS”
→ ✅ Put in VPC

---

👉 “Lambda needs internet from VPC”
→ ✅ NAT Gateway

---

👉 “Need fixed outbound IP”
→ ✅ NAT + Elastic IP

---

👉 “Must not lose messages”
→ ✅ SQS

---

👉 “Process immediately at scale”
→ ✅ SNS

---

# 🧩 Final Mental Model

```text id="m7y9kc"
VPC → private access
NAT → internet
SNS → speed
SQS → reliability
```
