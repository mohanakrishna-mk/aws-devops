# 🧠 Advanced Insight (What most miss)

👉 Strategy 2 beats Strategy 3 in most real systems

Why?

| Factor      | Winner           |
| ----------- | ---------------- |
| Control     | ALB (Strategy 2) |
| Speed       | ALB              |
| Reliability | ALB              |
| Simplicity  | ALB              |

👉 DNS-based (Strategy 3) is **less predictable**
