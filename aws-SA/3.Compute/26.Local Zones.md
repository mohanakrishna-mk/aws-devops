# 🧠 What are AWS Local Zones?

AWS Local Zones

👉 Local Zones = **mini AWS zones placed closer to cities**

```text id="6h2f8q"
AWS Region → extended into city → Local Zone
```

---

# ⚡ Core Idea

👉 Reduce latency **without leaving AWS network**

* Still part of a region
* Just geographically closer to users

## 🔹 Flow:

1. Region (e.g., us-east-1)
2. Extended to city (e.g., Boston)
3. You deploy resources there

---

## 🔑 Key Insight

```text id="0w5x6y"
Same AWS region → closer physical location
```

---

# ⚙️ What You Can Run

* Amazon EC2
* Amazon EBS
* Amazon RDS
* Amazon ECS
* Amazon ElastiCache

---

👉 More services than Wavelength
👉 Less than full region

---

# 🧠 Why Use Local Zones

---

## ⚡ Low Latency (City-level)

👉 Users in same metro area

---

## 🎯 Use Cases

* Gaming
* Media rendering
* Video streaming
* Virtual desktops
* Real-time apps (but not ultra-low like 5G)

---

# ⚙️ How You Use It (Important)

👉 You must:

1. Enable Local Zone
2. Extend VPC
3. Create subnet in Local Zone
4. Launch resources

---

👉 Key idea:

```text id="3d1v9l"
Local Zone = another AZ in your VPC
```

# ⚖️ Local Zones vs Wavelength vs Outposts

This is where exam questions come.

---

## 🟣 Local Zones vs Wavelength

| Feature  | Local Zones  | Wavelength            |
| -------- | ------------ | --------------------- |
| Target   | City users   | 5G mobile users       |
| Network  | AWS network  | Telecom network       |
| Latency  | Low          | Ultra-low             |
| Use case | General apps | Real-time mobile apps |

---

## 🔵 Local Zones vs Outposts

AWS Outposts

| Feature   | Local Zones       | Outposts          |
| --------- | ----------------- | ----------------- |
| Location  | AWS-managed city  | Your data center  |
| Purpose   | Latency reduction | Hybrid cloud      |
| Ownership | AWS               | Customer location |

---

## 🟢 Local Zones vs CloudFront

Amazon CloudFront

| Feature | Local Zones | CloudFront    |
| ------- | ----------- | ------------- |
| Type    | Compute     | CDN           |
| Purpose | Run apps    | Cache content |

---

# 🎯 Decision Rule (Exam Gold)

```text id="j8x3kp"
City latency → Local Zones
5G latency → Wavelength
On-prem AWS → Outposts
Caching → CloudFront
```
