# 🔹 What is S3 Storage Lens?

👉 **S3 Storage Lens = Analytics across ALL your S3 usage (organization-wide)**

Built on top of Amazon S3, it helps you:

* Understand storage usage
* Detect anomalies
* Optimize cost
* Improve security

📌 Works across:

* Accounts
* Regions
* Buckets
* Prefixes 

---

# 🔹 Key Features

## ✅ Organization-wide visibility

* Centralized view of all S3 usage
* Aggregates data across:

  * AWS Organization
  * Multiple accounts

---

## ✅ Default Dashboard

* Automatically created (no setup)
* Shows:

  * Total storage
  * Object count
  * Avg object size
  * Bucket distribution

📌 Cannot delete → only disable 

---

## ✅ Custom Dashboards

* Filter by:

  * Region
  * Account
  * Bucket
  * Storage class

---

## ✅ Export Reports

* Export to S3 bucket
* Formats:

  * CSV
  * Parquet 

---

# 🔹 Types of Metrics (Very Important)

## 1. 📊 Summary Metrics

* Total storage (bytes)
* Object count

👉 Use:

* Find unused or fast-growing buckets

---

## 2. 💰 Cost Optimization Metrics

* Incomplete multipart uploads
* Non-current versions

👉 Use:

* Clean unused storage
* Reduce costs

---

## 3. 🔐 Data Protection Metrics

* Versioning enabled?
* MFA delete enabled?
* Encryption (KMS)?

👉 Use:

* Identify insecure buckets

---

## 4. 🔑 Access Management Metrics

* Object ownership settings

---

## 5. ⚡ Event Metrics

* S3 event notifications usage

---

## 6. 🚀 Performance Metrics

* Transfer Acceleration usage

---

## 7. 📈 Activity Metrics

* GET / PUT requests
* Bytes downloaded

---

## 8. 🌐 HTTP Metrics

* Status codes (200, 403, etc.)

👉 Helps debug access issues

# 🔥 Storage Lens vs Storage Class Analysis

| Feature       | Storage Lens   | Storage Class Analysis |
| ------------- | -------------- | ---------------------- |
| Scope         | Entire org     | Single bucket          |
| Purpose       | Full analytics | Optimize storage class |
| Metrics       | Many types     | Access patterns only   |
| Visualization | Dashboard      | CSV reports            |

---

# 🧠 Simple Memory Trick

👉 **Storage Lens = Big Picture (All S3)**
👉 **Storage Class Analysis = Small Decision (per bucket)**
